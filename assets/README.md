# DeepLaw visual assets

These assets use an original geometric system built from source folios, version
checkpoints, bounded proof cards, and immutable receipts. They do not reuse the
Analytix symbol, wordmark geometry, plugin art, mascot art, or visual effects.
They intentionally avoid gavels, scales, shields, seals, national emblems, and
other imagery that could imply adjudication or official authority.

## Files

- `brand/deeplaw-mark.svg`: square source-to-receipt symbol.
- `brand/deeplaw-lockup.svg`: mark plus the DeepLaw working wordmark.
- `readme/evidence-execution-badge.svg`: compact value badge for repository
  surfaces; it makes no ranking or third-party endorsement claim.
- `readme/architecture.svg`: documentation view of the release and read-only
  evidence path. It is explanatory and is not runtime readiness evidence.

All SVGs are self-contained. They use no external fonts, scripts, links, or
embedded bitmap images. Internal CSS adapts text and surfaces to the viewer's
preferred light or dark color scheme. Each image includes an accessible
`title` and `desc`.
